git操作步骤 ： 
1、创建一个项目 jd 
在项目下创建项目结构
2、在项目目录下打开 git bash      初始化项目仓库（创建版本库）
   命令 ：    git init

3、使用 git status   查看哪些文件被git管理 
4、添加文件到暂存区    
添加一个文件 ：  git add 文件全称 
添加批量文件到暂存区 ： git add . 或 git add * 或 git add --all
			git add -A
5、将添加的文件提交到git服务器上 ：
git commit -m"日志信息"   日志信息必须要提供

6、提交的文件如果有被修改的，先添加 再提交
git add .
git commit -m""

7、git log  查看日志信息







使用git操作github上的项目
第一步 ：在github上创建一个项目仓库（版本库）
第二步 ： 克隆线上仓库路径到本地上  （在桌面上打开git bash 完成即可）
	git clone 仓库路径
第三步 ： 将要上传的项目 添加到 新克隆的版本库的工作区中
第四步 ： 在 新克隆的版本库中 重新打开git bash  ，在新的工作区中完成 添加 提交等操作
第五步 ： 将本地版本库中的项目提交到github上   ,这个过程需要提供 github的账号和密码
	命令 ：git push -v origin master
